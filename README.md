# OpenClaw Starter Subagent Bundle 🚀

Welcome to the **Free Open-Source Subagent Bundle** for OpenClaw! This package includes three foundational AI assistants ready to be deployed in your OpenClaw environment:

1. **📱 Social Media Manager** - Handles community interaction and content scheduling.
2. **📊 Data Analyst** - Processes CSV/JSON datasets and generates insights.
3. **💬 Customer Support Helpdesk** - Answers FAQs and manages support tickets based on your local knowledge base.

## Installation
Use the ClawHub CLI to install directly:
```bash
clawhub install abyss-kingdom-starter-bundle
```
